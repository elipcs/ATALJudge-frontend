
# AtalJudge Frontend

Este projeto é o frontend do sistema AtalJudge, criado com Next.js, TypeScript, Tailwind CSS e shadcn/ui.

## Principais tecnologias
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui

## Estrutura inicial
- src/app: páginas principais
- src/components: componentes reutilizáveis
- src/hooks: hooks customizados
- src/services: integração com APIs

## Como rodar

```bash
npm install
npm run dev
```
